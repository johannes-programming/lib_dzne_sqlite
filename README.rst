===================
lib_dzne_sqlite
===================

Overview
--------

Libary for handling sqlite. 

Installation
------------

To install lib_dzne_sqlite, you can use `pip`. Open your terminal and run:

.. code-block:: bash

    pip install lib_dzne_sqlite
